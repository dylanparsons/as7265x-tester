# AS7265x Spectral Sensor Test GUI

Professional test application for SparkFun AS7265x Spectral Triad sensors with multi-platform support.

**Author:** Dylan Parsons  
**Created:** August 2025  
**License:** MIT  
**Version:** 2.0.0

## Overview

This tool provides a professional-grade testing interface for AS7265x spectral sensors, designed to work across different hardware platforms (Raspberry Pi, development boards, etc.). Originally developed to fill the gap in available testing tools for this sensor family.

## Features

- ✅ **Multi-platform support** - Raspberry Pi, Arduino, custom boards
- ✅ **Professional GUI** - Easy-to-use desktop interface  
- ✅ **Automated testing** - I2C communication, device detection, sensor validation
- ✅ **Configurable platforms** - JSON-based hardware configuration
- ✅ **Production ready** - Suitable for QA, manufacturing, development
- ✅ **Open source** - MIT licensed for community benefit
